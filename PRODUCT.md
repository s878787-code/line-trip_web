# PRODUCT.md — TripRadar 車趟輔助(官網)

## 產品是什麼
TripRadar(車趟輔助)是台灣白牌司機的 LINE 接單助手 App:自動從大量 LINE 群組訊息辨識派單、整理成車趟卡片(上下車、距離、ETA),一鍵跳回群組標記派單人回覆。本 repo 是它的一頁式行銷官網(純靜態 HTML,GitHub Pages)。

## 目標受眾
40–60 歲職業司機(白牌、多群組接單)。用手機看網站居多、常在開車空檔快速瀏覽。
- 要:大字、高對比、直白口語、一眼看懂的價格
- 不要:術語、英文行話、小字灰字、需要思考的隱喻

## 網站唯一目標(轉換)
加官方 LINE(@493watvi)購買創始會員。所有設計服務這一件事。

## 品牌語氣
- 直接、口語、司機聽得懂:「少空等、多跑趟」「車子有跑,就有錢」
- 誠實:不保證收入、不宣稱 100% 不漏單、實機限制照實說
- 急迫但真實:限量與價格階梯是真的,不捏造數字

## 視覺系統(已定案,勿改方向)
- 深色編輯式:背景 `#0E1013`、表面 `#15181D`/`#1B1F25`、1px 實線邊框
- 品牌琥珀 `#FFB23E`(hover `#F5991C`)只當精準點綴;LINE 綠 `#06C755` 只用於 LINE 相關元素
- 字體:Sora(display)+ Inter + Noto Sans TC;數字 tabular-nums
- 圓角克制:10px / 8px 兩級;動效 150–300ms、cubic-bezier(.4,0,.2,1)
- 招牌元素:手機演示(LINE 訊息 → 雷達掃描 → 車趟卡 → 綠色回覆泡泡)

## Anti-references(絕對不要)
- 漸層文字、玻璃擬態/毛玻璃、發光陰影、漸層髮絲邊框、紫色漸層
- hover 浮起+放大、過度圓角(>12px)、彩色極光背景
- 「提升生產力」式空話文案、AI 罐頭感排版
- 假稀缺數據(假倒數、假名額)

## 紅線(公開 repo)
不得出現:授權金鑰、住家地址、真實客戶派單地址、真實 LINE 群組畫面(群名/暱稱/電話)。示範素材一律用模擬派單。
